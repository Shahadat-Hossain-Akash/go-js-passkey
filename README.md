# Go JS Passkey

A full-stack application using **Go** for backend services and **JavaScript** for the frontend. This project demonstrates modern backend practices including structured logging, environment-based config, PostgreSQL integration, and modular code structure.

---

## 🚀 Features

- ⚙️ Go HTTP server with static file serving
- 🔐 Passkey-style secure architecture (backend focus)
- 🧾 Custom logger to stdout and log file
- 🗃️ PostgreSQL database connection using environment variables
- 🧱 Clean, modular project structure
- ✅ Supports `.env` config
- 🔄 Optional live reloading with `air`
